# Patch Proposals

Store proposed skill patches here. These are suggestions only;
do not auto-apply them to core skills.

Expected naming: `<target-skill>-<short-label>.md`.

Patches document improvements to existing skills:
- Edge cases the current skill misses
- New sub-patterns discovered during solving
- Corrections to phase ordering
- Tool selection guidance improvements

Review periodically and promote matured patches into the
corresponding `skills/<category>/` directory.

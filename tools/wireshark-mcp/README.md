# Wireshark MCP

Local MCP wrapper for Wireshark desktop launch and tshark/capinfos packet analysis.

## Tools
- wireshark_status
- wireshark_list_interfaces
- wireshark_capinfos
- wireshark_tshark_summary
- wireshark_tshark_follow
- wireshark_open_capture

## Notes
- Prioritizes safe offline analysis of capture files.
- Live capture support is intentionally not exposed in this first version.
- Capture files must stay inside the configured workspace.
